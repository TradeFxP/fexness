# TradeFxP - Demo UI

This is a minimal Vite + React + Tailwind scaffold demonstrating:

- Single-file theme control: edit `src/theme.js` to change global colors (CSS variables).
- Tailwind for styling and responsive layout.
- Sticky header, slider, login & register pages, routing via `react-router-dom`.

Getting started (macOS / zsh):

```bash
cd /Users/vijaysonwane/Desktop/WB-1
npm install
npm run dev
```

Notes:
- Tailwind CSS is used; the CSS directives are in `src/index.css`.
- Change colors centrally by editing `src/theme.js` and reloading the app.
